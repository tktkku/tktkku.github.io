一段练习程序

2021-07-03 Views 漏洞239字2 min read繁 
// test.c
#include<stdio.h>
#include<string.h>

void main(void)
{
	char passwd[] = "password";
	char str[16];
	
	scanf("Input passwd:%s", &str);
	
	if (!strcmp(passwd, str))
	{
		printf("Correct!\n");
		return;
	}
	printf("Invalid!\n");
}

编译为test:

gcc -g -m32 -O0 -fno-stack-protector -z execstack -o test test.c

解释一下编译参数：

  • -g:在可执行文件中加入源码信息
  • -m32:使用32位编译
  • -O0:关闭所有优化
  • -fno-stack-protector:关闭栈保护
  • -z execstack:启用栈上代码可执行

利用LD_PRELOAD

LD_PRELOAD环境变量可以定义在程序运行前优先加载的动态链接库。因此我们可以刻意地构造一些相同名字的函数来劫持程序执行。

// hack.c
#include<stdio.h>

int strcmp(const char *s1, const char *s2)
{
	printf("Hacked!\n");
	return 0;
}

将hack.c编译为so文件：

gcc -shared -fPIC -m32 -o hack.so hack.so
  • -shared:
  • -fPIC:

运行时加载hack.so:

LD_PRELOAD="./hack.so" ./test
