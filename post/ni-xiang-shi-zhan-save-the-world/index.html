逆向实战-Save The World

2021-07-03 Views 逆向814字5 min read繁

工具

  • IDA
  • HIEW
  • 010Editor or WinHex

分析

示例程序1 Save The World.exe来自http://www.reversemes.de

先用EXEInfo查壳

双击运行，有一个弹窗，大意是说将这个弹窗改成一个密码验证窗口。

拖进IDA。

从401136开始都是创建编辑框。

.text:00401136 6A 00 push 0 ; lpParam
.text:00401138 FF 35 F4 33 40 00 push hInstance ; hInstance
.text:0040113E 6A 04 push 4 ; hMenu
.text:00401140 FF 75 08 push [ebp+hWndParent] ; hWndParent
.text:00401143 6A 35 push 35h ; '5' ; nHeight
.text:00401145 68 1C 02 00 00 push 21Ch ; nWidth
.text:0040114A 6A 05 push 5 ; Y
.text:0040114C 6A 00 push 0 ; X
.text:0040114E 68 04 08 00 50 push 50000804h ; dwStyle
.text:00401153 68 8D 30 40 00 push offset aYouHaveJustGai ; "You have just gained access to a server"...
.text:00401158 68 6A 30 40 00 push offset aEdit ; "edit"
.text:0040115D 6A 00 push 0 ; dwExStyle
.text:0040115F E8 FA 03 00 00 call CreateWindowExA

在4013DA处调用了lstrcmpA，比较是否与"Here is your mission..."相等。

.text:004013DA 68 00 02 00 00 push 200h ; nMaxCount
.text:004013DF 68 28 34 40 00 push offset String ; lpString
.text:004013E4 FF 35 00 34 40 00 push hWnd ; hWnd
.text:004013EA E8 8D 01 00 00 call GetWindowTextA
.text:004013EF 68 F7 31 40 00 push offset String2 ; "Here is your mission, if you choose to "...
.text:004013F4 68 28 34 40 00 push offset String ; lpString1
.text:004013F9 E8 5A 01 00 00 call lstrcmpA

目标其实很明确了，就是把其他的编辑框字符串不显示，让mission那里输入密码。

进行修改

用010Editor打开。

先点搜索字符串，找到那五句较长的。

定位到"You have gained..."，将其改为

定位到mission的前一句，将其改为

将"Here is your mission..."改为

此处就是要设置成的密码。

注意字符串后面要跟00，然后保存。双击运行。

其他的字符串未作处理，可以趁机将其他字符串前的字符覆盖为00，也可以修改汇编代码，使其不传进字符串参数。

直接在原本传入字符串的地方push 0，然后跟3个nop。

将密码那个编辑框的参数修改，宽度改成原来的一半，样式变成可以编辑的密码框。

50000804表示WS_CHILD | WS_VISIBLE | ES_READONLY | ES_MULTILINE；50000020表示WS_CHILD | WS_VISIBLE | ES_PASSWORD。

然后保存到文件。

运行看效果。

现在还需要对输入进行验证。

用Hiew打开，在4013FE处，按F3进入编辑模式，输入以下汇编

.text:004013FE ; ---------------------------------------------------------------------------
